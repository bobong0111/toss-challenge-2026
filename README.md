# 토스 디자이너 챌린지 2026 - 시간 조율 서비스 (Calendar & Coordination Dashboard)

토스 디자이너 챌린지 2026 과제 제출을 위한 시간 조율 서비스 웹 프로토타입 저장소입니다.  
사용자 친화적이고 직관적인 토스 고유의 디자인 시스템을 반영하여 설계 및 개발되었습니다.

👉 **[실시간 웹 프로토타입 확인하기 (GitHub Pages)]**(https://bobong0111.github.io/toss-challenge-2026/)

---

## 🌟 서비스 개요 (Service Overview)
"회의 시간 잡기가 협업의 가장 큰 병목이 되지 않도록."  
동료들의 캘린더 데이터를 자동으로 취합하고 분석하여 최적의 회의 시간을 찾고 조율하는 서비스입니다. 직관적인 UI 인터랙션과 토스의 깔끔한 디자인 언어를 바탕으로 설계되었습니다.

---

## 🌟 프로젝트 주요 특징 및 구현 기능

### 1. 개인 일정 관리 & 집중 시간 (Focus Time) 설정
* **주간 캘린더**: 주간 일정 현황을 한눈에 파악할 수 있는 그리드형 캘린더 인터페이스를 제공합니다.
* **집중 시간(Focus Time) 통합**: 마이페이지의 "회의 피하고 싶은 시간" 설정을 집중 시간으로 정립하여 보라색 테두리 및 배경(`#8f5bf6`)으로 시각화합니다. 
* **실시간 양방향 동기화**: 설정 변경 즉시 캘린더 그리드가 동기화되며, 직접 클릭 편집 시 경고 토스트 팝업으로 차단하여 데이터 무결성을 보호합니다.

### 2. 스마트 회의 개설 및 템플릿 위자드
* **회의 개설 위자드**: 회의 이름 입력, 회의 시간 선택, 참석자 검색 및 추가 등 회의 개설을 단계적으로 지원합니다.
* **템플릿 시스템**: 자주 사용하는 회의 형식(예: 1:1 면담 등)을 저장하고 재사용할 수 있는 카드형 템플릿 컴포넌트를 지원합니다.
* **참석자 검색 자동완성**: 이름 검색 시 실시간 매칭 목록을 제공하여 빠르고 간편하게 동료를 추가할 수 있습니다.

### 3. AI 기반 지능형 시간 추천 및 대시보드
* **캘린더 취합 분석**: 참석자 전원의 캘린더 일정을 취합하여 겹치지 않는 최적의 시간 슬롯을 분석합니다.
* **시간대 추천 카드**: AI가 분석한 우선순위별 최적 시간대(1순위, 2순위, 3순위)를 추천 카드로 시각화하여 제안합니다.
* **시간대별 상세 현황**: 캘린더의 특정 시간대를 클릭하면, 해당 시간에 누가 **'가능'**하고 **'불가'**한지 일관되고 직관적인 배지로 디테일 패널에 실시간 렌더링합니다.

### 4. 모두에게 물어보기 (투표 조율)
* **비동기 투표 수집**: 실시간 시간 조율이 불가능할 경우, 동료들에게 투표를 요청하여 의견을 취합할 수 있습니다.
* **투표 상태 트래킹**: 투표 중, 취합 완료 등 진행 상태에 따른 로딩 스피너 및 비주얼 피드백을 직관적으로 제공합니다.

### 5. 매끄러운 내비게이션 & UX 디테일
* **브라우저 히스토리 지원**: URL Hash 연동(`#landing`, `#dashboard`) 및 HTML5 History API 적용으로 브라우저 뒤로 가기 / 앞으로 가기 시 화면이 매끄럽게 동기화됩니다.
* **토스 스타일 다크/라이트 테마 변수**: 토스 고유의 서체 및 명도 대비가 우수한 모던 컬러 팔레트를 정적 CSS 변수로 정의하여 사용합니다.

## 🛠 사용 기술 스택
* **Frontend**: Vanilla HTML5, Vanilla CSS3 (Custom Variables, CSS Flexbox & Grid, Glassmorphism Backdrop-filter)
* **Logic**: Vanilla Javascript (HTML5 History API, KST Timezone helper, LocalStorage API)

## 🚀 로컬 실행 방법

별도의 복잡한 패키지 설치 없이 파이썬 내장 정적 서버를 이용해 바로 실행할 수 있습니다.

```bash
# 프로젝트 폴더로 이동 후 아래 명령 실행
python3 -m http.server 8000
```
이후 브라우저에서 `http://localhost:8000`으로 접속하여 즉시 테스트할 수 있습니다.
