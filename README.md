# 토스 디자이너 챌린지 2026 - 시간 조율 서비스 (Calendar & Coordination Dashboard)

토스 디자이너 챌린지 2026 과제 제출을 위한 시간 조율 서비스 웹 프로토타입 저장소입니다.  
사용자 친화적이고 직관적인 토스 고유의 디자인 시스템을 반영하여 설계 및 개발되었습니다.

👉 **[실시간 웹 프로토타입 확인하기 (GitHub Pages)]**(https://bobong0111.github.io/toss-challenge-2026/)

---

## 🌟 주요 구현 기능

### 1. 집중 시간 (Focus Time) 통합 관리 및 실시간 동기화
* **캘린더 자동 블록화**: 마이페이지 "회의 피하고 싶은 시간" 설정을 **집중 시간(Focus Time)** 개념으로 정립하고 보라색 테두리 및 배경(`#8f5bf6`)으로 직관적으로 렌더링합니다.
* **실시간 양방향 드로우**: 마이페이지에서 집중 시간을 추가/삭제하는 즉시 배경 캘린더 그리드가 새로고침 없이 동기화되어 그려집니다.
* **보안 및 예외 제어**: 캘린더 그리드 상에서 집중 시간 셀을 클릭해 직접 편집하려는 동작을 차단하고 경고 토스트 팝업을 표시합니다.

### 2. 외근 및 부재 일정 복원 및 렌더링 고도화
* **Wednesday 외근 복구**: 수요일 13시~16시의 "삼성역 미팅 (외근)" 일정을 자동 스캔 및 복원하여 캘린더 상에 주황색 오버레이로 표현합니다.
* **상세 패널 정보**: 시간 선택 상세 패널에서 해당 외근 일정 시간대를 확인할 때, 김토스의 상태를 Toss 스타일 오렌지색 "외근" 배지로 올바르게 연계하여 상세 표시합니다.

### 3. 히스토리 기반 브라우저 뒤로 가기 / 앞으로 가기 완벽 지원
* **URL Hash 연동**: 랜딩 페이지(`#landing`)와 추천 시간 대시보드(`#dashboard`)를 URL 해시 상태로 관리합니다.
* **HTML5 History API**: 브라우저 뒤로 가기 및 앞으로 가기 내비게이션 동작을 감지하여 화면이 유기적으로 동기화 전환되며, 비정상적인 화면 겹침 현상을 방지합니다.

### 4. 마이페이지 UI/UX 고도화 및 로그아웃 최적화
* **Toss 스타일 설정 카드**: 프로필 상세와 함께 비밀번호 변경, 프로필 수정, 로그아웃 등을 통합한 목록형 설정 컴포넌트로 개편했습니다.
* **즉시 로그아웃**: 마이페이지 내 로그아웃을 클릭하는 즉시, 로그인 창 전환과 동시에 열려있던 슬라이드 드로워 패널 및 백드롭 딤(dim) 오버레이를 즉시 완전히 숨겨 쾌적한 화면 전환을 완성합니다.

### 5. 토스트 팝업 문구 간소화 및 용어 통일
* 기존 "동료" 표현을 모두 **"참석자"**로 통일하여 협업 용어를 가다듬었습니다.
* 불필요하게 길거나 서술형인 토스트 문장들을 단수형 메시지로 줄여 간결성을 높였습니다.

## 🛠 사용 기술 스택
* **Frontend**: Vanilla HTML5, Vanilla CSS3 (Custom Variables, CSS Flexbox & Grid, Glassmorphism Backdrop-filter)
* **Logic**: Vanilla Javascript (HTML5 History API, KST Timezone helper, LocalStorage API)

## 🚀 로컬 실행 방법

별도의 복잡한 패키지 설치 없이 파이썬 내장 정적 서버를 이용해 바로 실행할 수 있습니다.

```bash
# 프로젝트 폴더로 이동 후 아래 명령 실행
python3 -m http.server 8000
```
이후 브라우저에서 `http://localhost:8000`으로 접속하여 즉시 테스트할 수 있습니다.
